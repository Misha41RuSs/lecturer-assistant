## [0.3.0] - 2026-04-24

### Added
- Основной набор unit-тестов для ключевых компонентов:
  - LectureBroadcastingBotTest — тесты Telegram бота
  - TestableBot — вспомогательный класс для тестирования бота
  - LectureServiceTest — тесты сервиса лекций
  - ExamServiceTest — тесты сервиса экзаменов
- Unit-тесты для GIFT парсера и других парсеров
- TestContainers фреймворк для интеграционного тестирования (реализация в процессе)
- Расширенный набор E2E тестов:
  - E2ELecturerFlowTest — полный поток лектора
  - LectureLoadSimulation — тесты нагрузочного тестирования
  - ContentServiceContractTest — тесты контрактов Content Service
  - IDORSecurityTest — тесты безопасности (IDOR уязвимости)
- Полное тестирование admin-gateway компонента
- Документирование найденных ошибок (BUGS_FOUND.md, BUGS.md)
- Документация E2E тестов (e2e-tests/REDME.md)
- Комплексная QA документация:
  - QA-3_Test-Lecture Broadcasting Service
  - QA-3_Test-Quiz-Service
  - QA-3_Test-Content_Service
  - QA3-testsAll_PowerShell — объединённый отчёт всех QA тестов

### Changed
- Актуализирована документация всех микросервисов
- Обновлены E2E тесты согласно новой функциональности
- Обновлены зависимости Maven (pom.xml) для поддержки TestContainers и e2e тестов
- Улучшена обработка ошибок в Content Service (логирование QA_3-errlog_1.txt)
- Консолидирована вся QA документация в единый отчёт

- Различные ошибки в интеграции компонентов, выявленные при тестировании
- Корректировка логирования ошибок (QA3-errlog_2 для Quiz Service)

### Infrastructure

## [0.1.0] - 2026-04-10

- Настроена Docker Compose инфраструктура с PostgreSQL
- API Gateway для единой точки входа всех сервисов

  - Парсеры для PDF и PPTX файлов
  - Контроллеры и сервисы для работы со слайдами
  - Хранилище файлов для загруженных презентаций
- Quiz Service для создания и проведения опросов
- Analytics Service для сбора и анализа данных студентов
- Frontend на React с поддержкой редактирования слайдов
- Electron приложение для десктопной версии
- Рисование на слайдах в реальном времени
- Система паролей для защиты лекций
- Загрузка презентаций (PowerPoint, PDF)
- Переключение между слайдами в реальном времени
- MVP функциональность подключения фронта к API
- Полная система экзаменов (бэкенд, фронт, бот, роутинг)
- Интеграция quiz-service и analytics-service во фронт
- Начало реализации Exam модели
- Реализованные REST API endpoints:
  - Создание лекции
  - Запуск лекции
  - Смена слайда
  - Загрузка презентации
- Базовые конфигурационные файлы (.env.sample, .env.example, .gitignore)

- PostgreSQL как основная база данных
- Docker контейнеризация всех компонентов

- GitHub для версионирования и управления исходным кодом

## История разработки

Проект «Цифровой ассистент лектора» начал разработку в марте 2026 года с планирования архитектуры. Первоначально была определена микросервисная архитектура с использованием Spring Boot и Docker.

В апреле 2026 года был реализован MVP с Electron приложением и WebSocket поддержкой для синхронизации в реальном времени, Telegram интеграция для управления лекциями, реализованы все основные микросервисы (Content, Quiz, Analytics, Lecture Broadcasting), а также полная система оценивания с импортом GIFT формата.

На этапе активной разработки (апрель 2026) была реализована система индивидуальных тестов для студентов, функции деанонимизации и исключения студентов, система рассылки опросов и аналитики.

На этапе тестирования и контроля качества (апрель 2026) был добавлен комплекс unit-тестов для всех основных компонентов (Bot, Service, Exam), e2e тесты для критических путей системы включая тесты безопасности (IDOR), нагрузочное тестирование и интеграция TestContainers. Все найденные ошибки задокументированы в BUGS.md с подробными QA отчётами.
